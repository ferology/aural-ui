## How to Use Themes

Each theme is a separate CSS file that you include in your project:

```html
<!-- Include the base Aural UI CSS -->
<link rel="stylesheet" href="path/to/aural-ui.css">

<!-- Include your chosen theme -->
<link rel="stylesheet" href="path/to/themes/minimal.css">
```

### Switching Themes Dynamically

You can switch themes at runtime by changing the theme stylesheet:

```javascript
function setTheme(themeName) {
  const themeLink = document.querySelector('link[href*="themes/"]');
  themeLink.href = `/themes/${themeName}.css`;
}

// Switch to dark theme
setTheme('dark');
```

## Design Tokens

All themes use the same CSS custom property names, making switching seamless:

```css
/* Colors */
--color-primary
--color-bg-primary
--color-text-primary
--color-border-subtle

/* Spacing */
--space-1 through --space-12

/* Typography */
--text-xs through --text-4xl

/* And many more... */
```

Each theme simply provides different values for these tokens.
