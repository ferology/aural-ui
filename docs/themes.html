Theme Guide

Aural UI provides flexible theming through CSS custom properties. Choose from pre-built themes or create your own to match your brand identity.

On This Page
Pre-built Themes Custom Themes Brand Theming Dynamic Switching

Pre-built Themes

Aural UI includes two professionally designed themes out of the box: Dark and Light. Both are optimized for accessibility and visual appeal.

Dark Theme

Optimized for low-light conditions

Dark Theme
Perfect for modern applications. Reduces eye strain in low-light environments.
<link rel="stylesheet" href="dark.css">

Light Theme

Clean and professional appearance

Light Theme
Traditional and professional. Great for daytime use and print-friendly interfaces.
<link rel="stylesheet" href="light.css">

Creating Custom Themes

Build your own custom theme by overriding CSS custom properties. Start with a base theme and customize the colors, spacing, and other tokens.

Step 1: Load a Base Theme

<!-- Load dark or light theme as your base --> <link rel="stylesheet" href="./dark.css">

Step 2: Override Design Tokens

<style> :root { /* Override primary color */ --color-primary: #8B5CF6; /* Purple */ --color-primary-hover: #7C3AED; --color-primary-alpha: rgba(139, 92, 246, 0.1); /* Override border radius for rounder components */ --radius-md: 1rem; --radius-lg: 1.5rem; /* Customize spacing */ --space-4: 1.25rem; /* Increase base spacing */ } </style>
Pro Tip: Use the Design Tokens reference to see all available customization options.

Brand Theming Examples

Create a theme that matches your brand identity. Here are examples for common brand color schemes.

Example: Tech Startup Theme (Blue/Purple)

:root { /* Primary: Vibrant Purple */ --color-primary: #8B5CF6; --color-primary-hover: #7C3AED; --color-primary-alpha: rgba(139, 92, 246, 0.1); /* Secondary: Electric Blue */ --color-secondary: #3B82F6; --color-secondary-hover: #2563EB; /* Modern rounded corners */ --radius-md: 0.75rem; --radius-lg: 1rem; }

Example: Finance/Corporate Theme (Blue/Navy)

:root { /* Primary: Professional Blue */ --color-primary: #1E40AF; --color-primary-hover: #1E3A8A; --color-primary-alpha: rgba(30, 64, 175, 0.1); /* Conservative styling */ --radius-md: 0.375rem; /* Less rounded */ --radius-lg: 0.5rem; /* Subtle shadows */ --shadow: 0 2px 4px 0 rgba(0, 0, 0, 0.05); }

Example: E-commerce Theme (Orange/Warm)

:root { /* Primary: Energetic Orange */ --color-primary: #F59E0B; --color-primary-hover: #D97706; --color-primary-alpha: rgba(245, 158, 11, 0.1); /* Success: Warm Green for purchases */ --color-success: #10B981; /* Friendly rounded corners */ --radius-md: 1rem; --radius-lg: 1.5rem; }

Dynamic Theme Switching

Allow users to switch between themes at runtime. Here's a simple implementation using JavaScript.

<!-- HTML --> <link rel="stylesheet" href="./dark.css" id="theme-link"> <button onclick="toggleTheme()">Toggle Theme</button> <!-- JavaScript --> <script> function toggleTheme() { const themeLink = document.getElementById('theme-link'); const currentTheme = themeLink.getAttribute('href'); if (currentTheme.includes('dark')) { themeLink.setAttribute('href', './light.css'); localStorage.setItem('theme', 'light'); } else { themeLink.setAttribute('href', './dark.css'); localStorage.setItem('theme', 'dark'); } } // Load saved theme preference const savedTheme = localStorage.getItem('theme'); if (savedTheme) { document.getElementById('theme-link') .setAttribute('href', `./${savedTheme}.css`); } </script>
Try it now! Click the "Toggle Theme" button in the bottom right corner to see theme switching in action.

Best Practices

Do

  • Test themes in both light and dark modes
  • Ensure WCAG AA contrast standards (4.5:1)
  • Save user theme preferences
  • Use semantic token names
  • Document custom theme tokens

Don't

  • Use hardcoded color values
  • Override base component styles directly
  • Create themes with poor contrast
  • Forget to test on mobile devices
  • Mix multiple theming approaches
