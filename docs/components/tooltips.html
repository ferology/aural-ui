Tooltips

Contextual help text that appears on hover or focus. Tooltips provide additional information without cluttering the interface.

Tooltip Positions

Tooltips appear in four positions. Use the data-tooltip attribute with data-tooltip-position:

<button data-tooltip="Help text" data-tooltip-position="top">
  Hover me
</button>
Icon Buttons with Tooltips

Tooltips are essential for icon-only buttons to explain their function. Always include both data-tooltip and aria-label for accessibility.

Keyboard Shortcuts

Use tooltips to show keyboard shortcuts and additional context for actions.
