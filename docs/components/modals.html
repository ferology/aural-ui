Best Practices

When to Use

  • Use modals for important decisions that require user confirmation
  • Use modals to collect focused input without leaving the current context
  • Use modals for critical information that must be acknowledged
  • Use modals for complex workflows that need isolation from the main content

Modal Sizes

  • .modal-sm - Use for simple confirmations and brief messages
  • Default size - Use for standard forms and content
  • .modal-lg - Use for complex forms or detailed content
  • .modal-fullscreen - Use for immersive experiences

Accessibility

  • Always include a visible close button with proper aria-label
  • Use role="dialog" and aria-modal="true" on the modal
  • Trap focus within the modal while it's open
  • Allow closing with the Escape key
  • Return focus to the trigger element when closed

JavaScript API

  • Aural.openModal('modal-id') - Opens a modal by ID
  • Aural.closeModal('modal-id') - Closes a modal by ID

1. Introduction

This modal demonstrates the scrollable variant, where the header and footer remain fixed while the body content scrolls independently.

2. Content Organization

The scrollable modal maintains visual hierarchy and keeps important actions always visible at the bottom.

  • Providing clear navigation context with a fixed header
  • Ensuring call-to-action buttons are always accessible
  • Allowing users to scroll through content comfortably

3. Use Cases

Scrollable modals are ideal for:

  • Legal Documents: Terms of service, privacy policies
  • Long Forms: Multi-section forms that require scrolling
  • Documentation: Help articles, guides, tutorials
  • Product Details: Detailed product information

4. Technical Implementation

The scrollable modal uses CSS flexbox with specific overflow properties:

  • The modal container uses display: flex and flex-direction: column
  • Header and footer have fixed dimensions
  • Body section has flex: 1 and overflow-y: auto

5. Conclusion

The scrollable modal pattern provides an elegant solution for presenting lengthy content within a modal dialog.
