When to Use

  • Activity Feed: Show recent system and user activity
  • Alerts: Centralize important alerts and warnings
  • Social Updates: Display comments, mentions, and interactions
  • Task Updates: Show progress on background tasks
  • System Messages: Communicate maintenance, updates, or announcements

Accessibility

  • Use role="region" and aria-label for the notification center
  • Announce new notifications with aria-live="polite"
  • Ensure keyboard navigation through notification items
  • Provide clear focus indicators on interactive elements
  • Use descriptive text, not just icons, for notification types
  • Allow dismissal via keyboard (Delete or Escape)
  • Announce notification count changes to screen readers

Best Practices

Visual Hierarchy

  • Use color-coded icons to indicate notification type (error, warning, success, info)
  • Clearly distinguish read from unread notifications
  • Show unread count in the header or badge
  • Group notifications by time or category

Content

  • Keep notification titles short and actionable
  • Provide enough context in the message
  • Include relative timestamps (e.g., "2 hours ago")
  • Show who triggered the notification when relevant

Interaction

  • Allow marking notifications as read/unread
  • Provide "Mark all as read" functionality
  • Enable notification settings access
  • Support inline actions where appropriate
  • Allow dismissing or archiving notifications

Performance

  • Implement pagination or virtual scrolling for long lists
  • Lazy load notification details
  • Consider batching multiple updates
  • Cache read state locally for faster UI updates

Common Mistakes to Avoid

  • Not providing a way to mark notifications as read
  • Missing empty state when no notifications exist
  • Poor contrast between read and unread states
  • Not grouping or organizing notifications logically
  • Forgetting to handle very long notification lists
  • Not providing notification preferences
  • Using only color to indicate notification type

Framework Examples

class NotificationCenter {
  constructor(containerId) {
    this.container = document.getElementById(containerId);
    this.notifications = [];
    this.init();
  }

  init() {
    this.render();
    this.setupEventListeners();
  }

  setupEventListeners() {
    this.container.addEventListener('click', (e) => {
      const item = e.target.closest('.aural-notification-center__item');
      if (item) {
        const index = Array.from(item.parentNode.children).indexOf(item);
        this.handleNotificationClick(index);
      }

      const clearAll = e.target.closest('[data-action="clear-all"]');
      if (clearAll) {
        this.clearAll();
      }

      const markAllRead = e.target.closest('[data-action="mark-all-read"]');
      if (markAllRead) {
        this.markAllAsRead();
      }
    });
  }

  addNotification(notification) {
    this.notifications.unshift({
      id: Date.now(),
      timestamp: new Date(),
      read: false,
      ...notification
    });
    this.render();
  }

  handleNotificationClick(index) {
    const notification = this.notifications[index];
    if (!notification.read) {
      notification.read = true;
      this.render();
    }

    // Execute notification action if provided
    if (notification.action) {
      notification.action();
    }
  }

  markAsRead(id) {
    const notification = this.notifications.find(n => n.id === id);
    if (notification) {
      notification.read = true;
      this.render();
    }
  }

  markAllAsRead() {
    this.notifications.forEach(n => n.read = true);
    this.render();
  }

  remove(id) {
    this.notifications = this.notifications.filter(n => n.id !== id);
    this.render();
  }

  clearAll() {
    this.notifications = [];
    this.render();
  }

  getUnreadCount() {
    return this.notifications.filter(n => !n.read).length;
  }

  formatTimestamp(date) {
    const diff = Date.now() - date.getTime();
    const minutes = Math.floor(diff / 60000);
    const hours = Math.floor(minutes / 60);
    const days = Math.floor(hours / 24);

    if (minutes < 1) return 'Just now';
    if (minutes < 60) return `${minutes} minute${minutes > 1 ? 's' : ''} ago`;
    if (hours < 24) return `${hours} hour${hours > 1 ? 's' : ''} ago`;
    return `${days} day${days > 1 ? 's' : ''} ago`;
  }

  render() {
    const unreadCount = this.getUnreadCount();

    this.container.innerHTML = `
      <div class="aural-notification-center">
        <div class="aural-notification-center__header">
          <h3 class="aural-notification-center__title">Notifications</h3>
          ${unreadCount > 0 ? `<span class="badge badge-primary">${unreadCount}</span>` : ''}
          <button class="btn btn-ghost btn-sm" data-action="mark-all-read">
            Mark all read
          </button>
        </div>
        <div class="aural-notification-center__list">
          ${this.notifications.length === 0 ? `
            <div class="empty-state">
              <div class="empty-state-icon">
                <i data-lucide="bell-off"></i>
              </div>
              <h3 class="empty-state-title">No notifications</h3>
              <p class="empty-state-description">You're all caught up!</p>
            </div>
          ` : this.notifications.map(n => `
            <button class="aural-notification-center__item ${!n.read ? 'aural-notification-center__item--unread' : ''} ${n.type ? `aural-notification-center__item--${n.type}` : ''}">
              <div class="aural-notification-center__item-icon">
                <i data-lucide="${n.icon}"></i>
              </div>
              <div class="aural-notification-center__item-content">
                <div class="aural-notification-center__item-title">${n.title}</div>
                <div class="aural-notification-center__item-message">${n.message}</div>
                <div class="aural-notification-center__item-time">${this.formatTimestamp(n.timestamp)}</div>
              </div>
              ${!n.read ? '<div class="aural-notification-center__item-dot"></div>' : ''}
            </button>
          `).join('')}
        </div>
        <div class="aural-notification-center__footer">
          <a href="#" class="aural-notification-center__footer-link">View all notifications</a>
        </div>
      </div>
    `;

    // Re-initialize Lucide icons
    if (window.lucide) lucide.createIcons();
  }
}
