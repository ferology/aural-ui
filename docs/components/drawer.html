Drawers can slide in from any edge of the screen. Use the position modifier class to specify the direction.

Navigation Pattern

Navigation drawers typically appear from the left and contain:

  • Navigation links with icons
  • User information at the top
  • Logout action in the footer
Settings Pattern

Settings drawers typically appear from the right and contain form controls, switches, and action buttons.

Mobile Action Sheet

Bottom drawers work well as mobile action sheets for quick options and share menus.

Best Practices

When to Use

  • Navigation menus (left drawer)
  • Shopping carts and details panels (right drawer)
  • Announcements and search (top drawer)
  • Action sheets and quick options (bottom drawer)

Position Guidelines

  • .aural-drawer--left - Primary navigation, menus
  • .aural-drawer--right - Settings, details, carts
  • .aural-drawer--top - Announcements, search bars
  • .aural-drawer--bottom - Mobile action sheets

Structure

  • Use .aural-drawer__header for title and close button
  • Use .aural-drawer__body for main content
  • Use .aural-drawer__footer for actions
  • Use .aural-drawer__nav for navigation links

Accessibility

  • Include role="dialog" and aria-modal="true"
  • Use aria-labelledby to reference the drawer title
  • Provide a visible close button with aria-label
  • Trap focus within the drawer when open
  • Close with Escape key

JavaScript API

  • Aural.openDrawer('drawer-id') - Opens a drawer
  • Aural.closeDrawer('drawer-id') - Closes a drawer

Framework Examples

<!-- HTML -->
<button class="btn btn-primary" onclick="openDrawer()">Open Drawer</button>

<div class="drawer-overlay" id="my-drawer">
    <div class="drawer drawer-right">
        <button class="drawer-close" onclick="closeDrawer()" aria-label="Close drawer">
            <i data-lucide="x"></i>
        </button>
        <div class="drawer-header">
            <h2 class="drawer-title">Drawer Title</h2>
        </div>
        <div class="drawer-body">
            <p>Drawer content goes here.</p>
        </div>
        <div class="drawer-footer">
            <button class="btn btn-ghost" onclick="closeDrawer()">Close</button>
            <button class="btn btn-primary" onclick="handleSave()">Save</button>
        </div>
    </div>
</div>

<script>
function openDrawer() {
    Aural.openDrawer('my-drawer');
}

function closeDrawer() {
    Aural.closeDrawer('my-drawer');
}

function handleSave() {
    console.log('Saving changes');
    closeDrawer();
}
</script>
import { useState } from 'react';

const Drawer = ({ isOpen, onClose, position = 'right', title, children, onSave }) => {
  if (!isOpen) return null;

  return (
    <div className="drawer-overlay">
      <div className={`drawer drawer-${position}`}>
        <button
          className="drawer-close"
          onClick={onClose}
          aria-label="Close drawer"
        >
          <i data-lucide="x"></i>
        </button>
        <div className="drawer-header">
          <h2 className="drawer-title">{title}</h2>
        </div>
        <div className="drawer-body">
          {children}
        </div>
        <div className="drawer-footer">
          <button className="btn btn-ghost" onClick={onClose}>
            Close
          </button>
          <button className="btn btn-primary" onClick={onSave}>
            Save
          </button>
        </div>
      </div>
    </div>
  );
};

// Usage
function App() {
  const [isDrawerOpen, setIsDrawerOpen] = useState(false);

  const handleSave = () => {
    console.log('Saving changes');
    setIsDrawerOpen(false);
  };

  return (
    <>
      <button
        className="btn btn-primary"
        onClick={() => setIsDrawerOpen(true)}
      >
        Open Drawer
      </button>

      <Drawer
        isOpen={isDrawerOpen}
        onClose={() => setIsDrawerOpen(false)}
        position="right"
        title="Drawer Title"
        onSave={handleSave}
      >
        <p>Drawer content goes here.</p>
      </Drawer>
    </>
  );
}
<template>
  <div>
    <button class="btn btn-primary" @click="isDrawerOpen = true">
      Open Drawer
    </button>

    <div v-if="isDrawerOpen" class="drawer-overlay">
      <div :class="`drawer drawer-${position}`">
        <button
          class="drawer-close"
          @click="closeDrawer"
          aria-label="Close drawer"
        >
          <i data-lucide="x"></i>
        </button>
        <div class="drawer-header">
          <h2 class="drawer-title">{{ title }}</h2>
        </div>
        <div class="drawer-body">
          <slot>
            <p>Drawer content goes here.</p>
          </slot>
        </div>
        <div class="drawer-footer">
          <button class="btn btn-ghost" @click="closeDrawer">
            Close
          </button>
          <button class="btn btn-primary" @click="handleSave">
            Save
          </button>
        </div>
      </div>
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      isDrawerOpen: false,
      title: 'Drawer Title',
      position: 'right'
    };
  },
  methods: {
    closeDrawer() {
      this.isDrawerOpen = false;
    },
    handleSave() {
      console.log('Saving changes');
      this.closeDrawer();
    }
  }
};
</script>
<script>
  import { Drawer, Button } from '@aural-ui/svelte';

    console.log('Saving changes');

    </Button>
    <Button variant="primary" on:click={handleSave}>
      Save
    </Button>
  </svelte:fragment>
</Drawer>

This is a left-side drawer. It slides in from the left edge of the screen.

Drawers are useful for:

  • Navigation menus
  • Filters and settings
  • Additional content panels
  • Contextual actions

Right Drawer

This is a right-side drawer. It slides in from the right edge of the screen.

Right drawers are commonly used for:

  • Shopping carts
  • Details panels
  • User profiles
  • Notifications

Top Drawer

This is a top drawer. It slides down from the top of the screen.

Useful for announcements, search bars, or temporary notifications.

Bottom Drawer

This is a bottom drawer. It slides up from the bottom of the screen.

Common on mobile for action sheets, share menus, or quick options.
