When to Use

  • Photo Collections: Display related images in an organized grid
  • Product Catalogs: Showcase products with hover information
  • Portfolios: Present work samples with project details
  • Team Pages: Display team member photos with names and roles
  • Media Libraries: Organize and browse image collections

Accessibility

  • Include tabindex="0" on gallery items for keyboard navigation
  • Provide meaningful alt text for all images
  • Ensure overlay content is accessible via keyboard focus
  • Use sufficient color contrast for overlay text
  • Consider adding aria-label for additional context
  • Ensure focus states are clearly visible
  • Don't hide essential information only in overlays

Best Practices

Grid Layout

  • Use aural-gallery--cols-2, --cols-3, or --cols-4 for different densities
  • Consider responsive breakpoints for mobile layouts
  • Maintain consistent aspect ratios for a clean grid
  • Use appropriate column counts based on content type

Images

  • Optimize images for web to ensure fast loading
  • Use consistent dimensions within a gallery
  • Provide appropriately sized images for different screen sizes
  • Consider lazy loading for large galleries

Overlays

  • Keep overlay content concise and scannable
  • Use consistent overlay styles across all items
  • Ensure text remains readable over varied image backgrounds
  • Don't rely solely on hover for critical information

Interaction

  • Add click handlers for lightbox or detail views
  • Provide clear hover and focus states
  • Consider touch-friendly alternatives for mobile
  • Support keyboard navigation through the grid

Common Mistakes to Avoid

  • Using images with inconsistent aspect ratios
  • Hiding important information only in hover states
  • Not providing keyboard accessibility
  • Using too many columns on small screens
  • Poor contrast between overlay text and images
  • Missing or generic alt text for images
  • Not optimizing images for performance
