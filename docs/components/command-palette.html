When to Use

  • Power User Features: Provide quick access to all application features
  • Navigation: Enable fast navigation to any page or section
  • Search: Unified search across files, commands, and content
  • Quick Actions: Execute common tasks without navigating menus
  • Settings Access: Quick theme switching and preference changes

Accessibility

  • Use role="dialog" and aria-modal="true" on the palette
  • Implement focus trapping within the command palette
  • Support keyboard navigation with arrow keys
  • Provide aria-activedescendant for the selected item
  • Use aria-label on the search input
  • Announce search results count to screen readers
  • Ensure Escape key closes the palette and returns focus

Best Practices

Keyboard Shortcuts

  • Use ⌘K / Ctrl+K as the standard shortcut to open
  • Display keyboard shortcuts for commands when available
  • Support arrow key navigation through results
  • Enter key should execute the selected command
  • Escape key should close the palette

Search & Filtering

  • Implement fuzzy search for better user experience
  • Show recent/frequent commands at the top
  • Group results by category for easier scanning
  • Highlight matching text in search results
  • Provide instant feedback as users type

Visual Design

  • Center the palette horizontally, positioned near the top
  • Use a semi-transparent backdrop to maintain context
  • Limit visible results to avoid overwhelming users
  • Show clear visual focus state on the active item
  • Use icons to help users quickly identify command types

Content Organization

  • Group related commands together
  • Provide descriptive subtitles for complex commands
  • Keep command titles short and action-oriented
  • Use consistent terminology throughout

Common Mistakes to Avoid

  • Not implementing proper keyboard navigation
  • Missing focus management when opening/closing
  • Slow search performance with large command sets
  • Not providing visual feedback for the selected item
  • Inconsistent keyboard shortcuts across platforms
  • Overcrowding the palette with too many options
  • Not closing the palette after command execution
