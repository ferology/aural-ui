Best Practices

When to Use

  • Use chips to represent tags, categories, or attributes
  • Use removable chips for filters that users can dismiss
  • Use chip inputs for adding multiple items like skills or interests
  • Use colored variants to indicate status or category types
  • Keep chip text concise - use 1-3 words maximum

Accessibility

  • Always include aria-label on remove buttons that describes what will be removed
  • The remove button has a minimum touch target of 44Ã—44px for accessibility
  • Use semantic color variants - success for positive, error for negative, etc.
  • Ensure sufficient color contrast between text and background
  • Provide keyboard support for chip input (Enter to add, Backspace to remove last)

Size Guidelines

  • .aural-chip--sm - Use in compact interfaces like tables or sidebars
  • Default size - Use for most standard UI contexts
  • .aural-chip--lg - Use for prominent tag displays or hero sections

Color Guidelines

  • .aural-chip--primary - Active or selected states
  • .aural-chip--success - Positive status (completed, approved)
  • .aural-chip--warning - Cautionary status (pending, in progress)
  • .aural-chip--error - Negative status (failed, blocked)
  • .aural-chip--info - Informational status (new, review)
  • Default (no color class) - Neutral tags and categories

Interactive Behaviors

  • Removable chips should show hover states on both the chip and remove button
  • Use onclick="this.parentElement.remove()" for simple removal
  • For chip inputs, press Enter to add a new chip from the input field
  • Consider maximum limits for chip inputs (e.g., max 10 tags)

const Chip = ({ children, variant, icon, removable, onRemove }) => {
  const classNames = [
    'aural-chip',
    variant && `aural-chip--${variant}`,
    removable && 'aural-chip--removable'
  ].filter(Boolean).join(' ');

  return (
    <div className={classNames}>
      {icon && <i data-lucide={icon} />}
      {children}
      {removable && (
        <button
          className="aural-chip__remove"
          onClick={onRemove}
          aria-label="Remove"
        >
          <i data-lucide="x" />
        </button>
      )}
    </div>
  );
};

<template>
  <div :class="chipClasses">
    <i v-if="icon" :data-lucide="icon" />
    <slot />
    <button
      v-if="removable"
      class="aural-chip__remove"
      @click="$emit('remove')"
      aria-label="Remove"
    >
      <i data-lucide="x" />
    </button>
  </div>
</template>

<script>
export default {
  props: {
    variant: String,
    icon: String,
    removable: Boolean
  },
  computed: {
    chipClasses() {
      return [
        'aural-chip',
        this.variant && `aural-chip--${this.variant}`,
        this.removable && 'aural-chip--removable'
      ].filter(Boolean).join(' ');
    }
  }
}
</script>
