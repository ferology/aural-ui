Input Sizes with Icons

Add icons to inputs using the .input-group wrapper with .input-group-icon elements. Use these classes to position icons:

  • .input-group-prefix - Adds padding for left icon
  • .input-group-suffix - Adds padding for right icon
  • .input-group-icon-left - Position icon on the left
  • .input-group-icon-right - Position icon on the right

Available sizes: .input-group-sm, default, and .input-group-lg

Textarea with Character Counter
Characters: 0 / 200

Character counters help users stay within input limits. The counter automatically updates as users type:

  • Default style when under 80% of limit
  • .input-counter-warning - Orange when 80-99% of limit
  • .input-counter-error - Red when at or over limit

Use the .input-with-counter wrapper to enable character counting.
