Alert Pattern

Alert dialogs use centered layouts with prominent icons to clearly communicate status. The icon's background color matches the alert type.

Decision Pattern

Decision dialogs present a clear question with two action buttons. The primary action should be on the right, with secondary/cancel action on the left.

Best Practices

Dialog vs Modal

  • Dialogs are compact modals for quick interactions
  • Use dialogs for simple messages, confirmations, and alerts
  • Use modals for complex forms or detailed content
  • Dialogs typically use .modal-sm size

Status Dialogs

  • Use success for completed actions
  • Use error for failed operations
  • Use info for informational messages
  • Use warning for cautionary notices

Button Placement

  • Single action: Use primary button centered or right-aligned
  • Two actions: Cancel/secondary on left, primary action on right
  • Destructive actions: Use .btn-error or .btn-danger

Accessibility

  • Always include a way to dismiss the dialog
  • Use role="alertdialog" for alerts that require acknowledgment
  • Focus should move to the first interactive element
  • Escape key should close the dialog

<script>
function openDialog() {
    Aural.openModal('my-dialog');
}

function closeDialog() {
    Aural.closeModal('my-dialog');
}

function handleConfirm() {
    console.log('Action confirmed');
    closeDialog();
}
</script>
import { useState } from 'react';

const Dialog = ({ isOpen, onClose, title, subtitle, message, onConfirm }) => {
  if (!isOpen) return null;

  return (
    <div className="modal-overlay">
      <div className="modal modal-sm">
        <button
          className="modal-close"
          onClick={onClose}
          aria-label="Close dialog"
        >
          <i data-lucide="x"></i>
        </button>
        <div className="modal-header">
          <div className="modal-icon">
            <i data-lucide="info"></i>
          </div>
          <div>
            <h2 className="modal-title">{title}</h2>
            {subtitle && <p className="modal-subtitle">{subtitle}</p>}
          </div>
        </div>
        <div className="modal-body">
          <p>{message}</p>
        </div>
        <div className="modal-footer">
          <button className="btn btn-ghost btn-sm" onClick={onClose}>
            Cancel
          </button>
          <button className="btn btn-primary btn-sm" onClick={onConfirm}>
            OK
          </button>
        </div>
      </div>
    </div>
  );
};

// Usage
function App() {
  const [isDialogOpen, setIsDialogOpen] = useState(false);

  const handleConfirm = () => {
    console.log('Action confirmed');
    setIsDialogOpen(false);
  };

  return (
    <>
      <button
        className="btn btn-primary"
        onClick={() => setIsDialogOpen(true)}
      >
        Open Dialog
      </button>

      <Dialog
        isOpen={isDialogOpen}
        onClose={() => setIsDialogOpen(false)}
        title="Confirm Action"
        subtitle="Are you sure?"
        message="This action requires confirmation."
        onConfirm={handleConfirm}
      />
    </>
  );
}

        <div class="modal-header">
          <div class="modal-icon">
            <i data-lucide="info"></i>
          </div>
          <div>
            <h2 class="modal-title">{{ title }}</h2>
            <p class="modal-subtitle">{{ subtitle }}</p>
          </div>
        </div>
        <div class="modal-body">
          <p>{{ message }}</p>
        </div>
        <div class="modal-footer">
          <button class="btn btn-ghost btn-sm" @click="closeDialog">
            Cancel
          </button>
          <button class="btn btn-primary btn-sm" @click="handleConfirm">
            OK

<script>
export default {
  data() {
    return {
      isDialogOpen: false,
      title: 'Confirm Action',
      subtitle: 'Are you sure?',
      message: 'This action requires confirmation.'
    };
  },
  methods: {
    closeDialog() {
      this.isDialogOpen = false;
    },
    handleConfirm() {
      console.log('Action confirmed');
      this.closeDialog();
    }
  }
};
</script>
<script>
  import { Modal, Button } from '@aural-ui/svelte';

  let isOpen = false;

  function handleConfirm() {
    console.log('Action confirmed');
    isOpen = false;
  }
</script>

<Button on:click={() => isOpen = true}>
  Open Dialog
</Button>

<Modal
  bind:open={isOpen}
  title="Confirm Action"
  subtitle="Are you sure?"
  size="small"
>
  <div slot="icon">
    <i data-lucide="info"></i>
  </div>

  <p>This action requires confirmation.</p>

  <svelte:fragment slot="footer">
    <Button variant="ghost" size="small" on:click={() => isOpen = false}>
      Cancel
    </Button>
    <Button variant="primary" size="small" on:click={handleConfirm}>
      OK
    </Button>
  </svelte:fragment>
</Modal>
