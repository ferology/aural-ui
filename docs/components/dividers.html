When to Use Dividers

Dividers help organize content and create visual hierarchy:

  • Content sections: Separate distinct sections of related content
  • List items: Divide items in lists, menus, or navigation
  • Form groups: Distinguish between different sections of a form
  • Toolbars: Group related actions in button groups or toolbars
  • Alternative flows: Show "OR" options in authentication or payment flows

Best Practices

Visual Hierarchy

  • Use dividers to establish clear visual hierarchy and content structure
  • Avoid overusing dividers - too many can create visual clutter
  • Consider using whitespace instead of dividers when separation is already clear
  • Use consistent divider styles throughout your application

Style Selection

  • Use solid dividers for primary content separation
  • Use dashed dividers for less prominent separations
  • Use dotted dividers for subtle visual breaks
  • Use dividers with text for alternative flows or optional sections

Spacing

  • Use compact spacing for densely packed content like lists
  • Use default spacing for standard content sections
  • Use spacious spacing for major section breaks
  • Maintain consistent spacing between divider and surrounding content

Orientation

  • Use horizontal dividers for vertically stacked content
  • Use vertical dividers for inline or horizontally arranged items
  • Match vertical divider height to the content it separates
  • Ensure vertical dividers are vertically centered with their content

Accessibility

While dividers are primarily visual, consider these accessibility points:

  • Use semantic HTML (<hr>) for horizontal dividers when they represent thematic breaks
  • Dividers should have sufficient contrast with the background (at least 3:1)
  • Don't rely solely on dividers to convey meaning - use headings and proper document structure
  • Screen readers may announce <hr> elements as "separator" or "horizontal rule"
  • For purely decorative dividers, consider using CSS borders instead of <hr>
  • When using text with dividers, ensure text has proper color contrast

Semantic Usage

<!-- Thematic break (semantic) -->
<hr class="divider">

<!-- Decorative only (CSS border) -->
<div style="border-bottom: 1px solid var(--color-border);"></div>

Mobile Considerations

  • Dividers remain effective on mobile devices for content organization
  • Consider using more generous spacing on mobile to improve readability
  • Vertical dividers may not work well on narrow mobile screens
  • Test divider visibility on various screen sizes and pixel densities
  • Be mindful of divider length in responsive layouts
