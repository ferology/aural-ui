Quick Start

Once you've included Aural UI in your project, you can start using components immediately. Here's a simple example:

1. Basic HTML Structure

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Aural UI App</title>

    <!-- Aural UI CSS -->
    <link rel="stylesheet" href="aural-ui.css">
    <link rel="stylesheet" href="dark.css" id="theme-link">
</head>
<body>
    <!-- Your content here -->
</body>
</html>

2. Add Your First Components

<div class="card" style="max-width: 400px; margin: 2rem auto; padding: var(--space-6);">
    <h2 style="margin: 0 0 var(--space-4) 0;">Welcome to Aural UI</h2>
    <p style="color: var(--color-text-secondary); margin-bottom: var(--space-4);">
        Start building beautiful interfaces with our accessible component library.
    </p>
    <button class="btn btn-primary">Get Started</button>
</div>

Available Themes

Aural UI comes with multiple built-in themes. Switch between them by changing the theme stylesheet:

  • Dark - Default dark theme with excellent contrast
  • Light - Clean light theme for bright environments
  • Neon - Vibrant cyberpunk-inspired theme
  • Kinetic - Modern theme with dynamic energy
  • Prismatic - Sophisticated gradient-based aesthetic
  • High Contrast - Maximum accessibility for low vision
  • Colorblind - Optimized for color vision deficiency
  • Minimal - Clean, distraction-free design
  • Warm - Inviting earth tones and warm colors
<!-- Change theme by swapping the stylesheet -->
<link rel="stylesheet" href="dark.css" id="theme-link">
<link rel="stylesheet" href="light.css" id="theme-link">
<link rel="stylesheet" href="neon.css" id="theme-link">
<link rel="stylesheet" href="kinetic.css" id="theme-link">
