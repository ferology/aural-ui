Step 1: Installation

Choose your preferred method to add Aural UI to your project. We recommend using the CDN for quick prototyping or NPM for production applications.

CDN Fastest
Perfect for quick prototyping and demos
NPM Recommended
Best for production applications
Download
Self-host the files on your server
  • 1
    Choose your installation method above
  • 2
    Copy the code from the preview panel
  • 3
    Add it to your HTML file's <head> section
  • 4
    You're ready to start building!
Installation Code
<!-- Aural UI CSS --> <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/aural-ui@1.0.0/dist/aural-ui.min.css"> <!-- Choose a theme (dark or light) --> <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/aural-ui@1.0.0/themes/dark.css"> <!-- Aural UI JavaScript --> <script src="https://cdn.jsdelivr.net/npm/aural-ui@1.0.0/dist/aural-ui.min.js"></script>
File Structure:
  • aural-ui.css - Core component styles
  • dark.css or light.css - Theme colors
  • aural-ui.js - Interactive component behaviors

Step 2: Your First Component

Let's create your first Aural UI component - a button! Buttons are the building blocks of user interfaces.

  • 1
    Add the .btn class to create a basic button
  • 2
    Use .btn-primary, .btn-secondary, or .btn-danger for different styles
  • 3
    Add .btn-sm or .btn-lg to change the size
  • 4
    Try adding icons using Lucide Icons for a professional look
Live Preview
<!-- Basic Button --> <button class="btn btn-primary"> Click Me </button> <!-- Button with Icon --> <button class="btn btn-primary"> <i data-lucide="heart"></i> Like </button> <!-- Different Sizes --> <button class="btn btn-primary btn-sm">Small</button> <button class="btn btn-primary">Medium</button> <button class="btn btn-primary btn-lg">Large</button>

Step 3: Form Components

Build beautiful, accessible forms using Aural UI's form components. Let's create a simple login form.

  • 1
    Use .input class for text inputs
  • 2
    Wrap labels and inputs in .form-group for proper spacing
  • 3
    Use .checkbox for checkboxes and .radio for radio buttons
  • 4
    Add .error class to show validation errors
Login Form
<form> <div class="form-group"> <label for="email">Email</label> <input type="email" id="email" class="input" placeholder="you@example.com"> </div> <div class="form-group"> <label for="password">Password</label> <input type="password" id="password" class="input" placeholder="••••••••"> </div> <div class="form-group"> <label class="checkbox"> <input type="checkbox"> <span>Remember me</span> </label> </div> <button type="submit" class="btn btn-primary"> Sign In </button> </form>

Step 4: Layout & Grid

Create responsive layouts using Aural UI's grid system and card components. Perfect for dashboards and content galleries.

  • 1
    Use .container to center and constrain content width
  • 2
    Use .grid with .grid-cols-* for responsive columns
  • 3
    Cards automatically adapt to the grid layout
  • 4
    Use .gap-* to control spacing between grid items
Responsive Card Grid
<div class="container"> <div class="grid grid-cols-3 gap-4"> <div class="card"> <h3>Card Title</h3> <p>Card content goes here</p> </div> <div class="card"> <h3>Card Title</h3> <p>Card content goes here</p> </div> <div class="card"> <h3>Card Title</h3> <p>Card content goes here</p> </div> </div> </div>

Step 5: Theming

Customize Aural UI to match your brand. Switch between pre-built themes or create your own custom theme using CSS variables.

  • 1
    Include either dark.css or light.css for pre-built themes
  • 2
    Override CSS variables to customize colors
  • 3
    Use the theme toggle button to test both themes
  • 4
    Create custom themes by defining your own color palette
Custom Theme Example
<style> /* Create a custom theme */ :root { /* Override primary color */ --color-primary: #8B5CF6; /* Purple */ --color-primary-hover: #7C3AED; --color-primary-alpha: rgba(139, 92, 246, 0.1); /* Override success color */ --color-success: #10B981; /* Green */ --color-success-hover: #059669; /* Override border radius */ --radius-md: 12px; --radius-lg: 16px; } </style>

  • Build responsive layouts
  • Customize themes
