v1.5.0 Latest February 28, 2026

New Features

  • Storybook Integration: Multi-theme component preview system with createThemeGrid utility for side-by-side theme comparison, AllThemes story showing components across all 9 themes, and ThemeComparison story with interactive controls
  • New Themes: Expanded from 7 to 9 themes
    • Minimal theme: Clean, understated design with subtle styling
    • Warm theme: Earthy tones with amber accents
    • Prismatic theme (replaces Neon Refined): Colorful prismatic design
  • Theme System Enhancements: High Contrast and Colorblind-Friendly themes now fully integrated with proper theme manager definitions
  • Accessibility Enhancement: Explanatory tooltip for "Skip to main content" links - helps users understand accessibility feature on first encounter with message: "Press Tab to see this link - allows keyboard users to skip navigation and jump directly to main content"

Changes

  • Documentation Updates: All 60+ documentation files updated to reflect 9 themes
  • Navigation Improvements: Theme Switcher relocated to bottom of sidebar for better UX, simplified navigation structure (removed Theme Showcases section)
  • Theme System: Neon Refined renamed to Prismatic theme, updated landing page with Prismatic, Minimal, and Warm theme cards
  • UI Polish: Search bar placeholder simplified (removed "components" word), theme manager now supports all 9 themes with proper definitions

Removed

  • "Try Themes Live" interactive section from themes.html (synchronization issues)
  • Theme Showcases navigation section (redundant with Themes documentation page)

Documentation Enhancements

  • Navigation Components: Transform 7 navigation pages (Breadcrumbs, Pagination, Tabs, Context Menu, Navbar, Stepper, Bottom Navigation) to comprehensive 4-tab structure with Overview, Variants, Patterns, and Guidelines
  • Interactive Components: Enhance 6 interactive component pages (Accordions, Dropdowns, Popovers, Modals, Dialog, Command Palette) with expanded examples, accessibility improvements, and collapsible code sections

  • Accessibility Guide: Expand accessibility.html with detailed CSS vs ARIA responsibilities and component-specific requirements

Accessibility Improvements

  • Add aria-label="Toggle theme" to all theme toggle buttons across documentation
  • Add aria-hidden="true" to 100+ decorative icons throughout components
  • Implement proper ARIA roles for context menus (role="menu", role="menuitem", role="menuitemcheckbox")
  • Add aria-controls and aria-labelledby to all tab panel relationships
  • Add descriptive aria-label attributes to all stepper step indicators
  • Ensure 44x44px minimum touch targets on all interactive elements for mobile accessibility

CSS Enhancements

  • Touch Device Optimizations: Add minimum 44px touch targets across all interactive components
  • Reduced Motion Support: Enhance prefers-reduced-motion media query support across all animated components
  • Focus States: Add proper :focus-visible states for keyboard navigation
  • Input Components: Standardize focus, hover, and disabled states across all form controls

Bug Fixes

  • Fix range slider HTML structure to match slider component for consistency
  • Fix range slider interactivity with correct CSS selectors
  • Fix file upload avatar button to use standard button classes
  • Fix styling inconsistencies across component documentation pages
v1.3.0 January 26, 2026

New Features

  • Enhanced Theme Selector: Replaced simple toggle with dropdown showing all 4 themes (Dark, Light, High Contrast, Colorblind-Friendly) with visual icons
  • High Contrast Theme: New WCAG AAA compliant theme with 7:1+ contrast ratios, pure black/white colors, and enhanced 3px focus outlines
  • Colorblind-Friendly Theme: Optimized for users with color vision deficiency using blue for success and orange for errors instead of red/green
  • 150+ Utility Classes: Added backdrop filters, gradients (8 directions), aspect ratios, dark mode variants, and RTL support

Component Enhancements

  • Icon-Only Buttons: Added .btn-icon class with 3 sizes (36px, 44px, 52px) for compact toolbar buttons
  • Input Icons: Support for prefix and suffix icons using .input-group wrapper with proper spacing
  • Character Counter: Live character counting for inputs/textareas with warning (80%) and error (100%) states
  • Expandable Table Rows: Click-to-expand rows with smooth animation and rotatingchevron icons
  • Fullscreen Modals: Added .modal-fullscreen for 100% viewport coverage
  • Scrollable Modals: Added .modal-scrollable with fixed header/footer and scrolling body

Documentation

  • Enhanced Buttons documentation with icon-only examples and accessibility tips
  • Enhanced Inputs documentation with icon examples and character counter guide
  • Enhanced Tables documentation with expandable row examples
  • Enhanced Modals documentation with fullscreen and scrollable variants
  • Added comprehensive Theme Guide documenting all 4 themes with specifications
  • Created Utilities reference with tables and examples for all 150+ utility classes
