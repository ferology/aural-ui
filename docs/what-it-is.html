What it is

Aural UI is a comprehensive design system that provides a complete set of UI components, design tokens, and patterns for building modern web applications. It's built with accessibility, customization, and developer experience at its core.

50+ Components

A complete library of production-ready components from buttons to complex data tables.

Accessibility First

Built with WCAG 2.1 AA standards, keyboard navigation, and screen reader support.

Fully Themeable

Customize every aspect with CSS custom properties and design tokens.
