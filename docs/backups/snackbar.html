Snackbar

Brief messages with optional actions, appearing at the edge of the screen.

Basic Snackbars
View Code 
// Default snackbar
Aural.showSnackbar('This is a default snackbar message');

// Success snackbar
Aural.showSnackbar('Operation completed successfully!', { type: 'success' });

// Error snackbar
Aural.showSnackbar('An error occurred!', { type: 'error' });
With Action Button
View Code 
Aural.showSnackbar('New message received', {
    description: 'From John Doe',
    type: 'info',
    action: { label: 'View', onClick: () => alert('Viewing message') }
});
All Types
View Code 
Aural.showSnackbar('Default message');
Aural.showSnackbar('Success message', { type: 'success' });
Aural.showSnackbar('Info message', { type: 'info' });

Position Variants
View Code 
Aural.showSnackbar('Bottom Left', { position: 'bottom-left' });
Aural.showSnackbar('Bottom Center', { position: 'bottom-center' });
Aural.showSnackbar('Bottom Right', { position: 'bottom-right' });
Aural.showSnackbar('Top Center', { position: 'top-center' });
Custom Duration
View Code 
Aural.showSnackbar('Quick message', { duration: 2000 });  // 2 seconds
Aural.showSnackbar('Extended message', { duration: 8000 });  // 8 seconds
Undo Pattern
View Code 
Aural.showSnackbar('Item deleted', {
    type: 'error',
    action: {
        label: 'UNDO',
        onClick: () => Aural.showSnackbar('Item restored', { type: 'success' })
    },
    duration: 6000
});

Snackbars with undo actions allow users to quickly reverse destructive actions.

Best Practices

Snackbar vs Toast

  • Snackbars can include action buttons
  • Snackbars typically appear at the bottom of the screen
  • Toasts are purely informational
  • Use snackbars when undo/retry actions are helpful

When to Use

  • Confirming user actions (delete, archive, move)
  • Providing undo opportunities
  • Quick status updates with optional actions
  • Network status changes

JavaScript API

  • Aural.showSnackbar(message) - Basic snackbar
  • Aural.showSnackbar(message, { type }) - With type
  • Aural.showSnackbar(message, { action }) - With action
  • Aural.showSnackbar(message, { position }) - Custom position

Options

  • type - 'default', 'success', 'error', 'warning', 'info'
  • description - Additional details
  • position - 'bottom-left', 'bottom-center', 'bottom-right', etc.
  • duration - Time in milliseconds
  • action - { label, onClick }

Accessibility

  • Snackbars are announced to screen readers
  • Actions should be keyboard accessible
  • Ensure sufficient contrast
  • Allow enough time to read and interact
