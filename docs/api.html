JavaScript API Reference

Complete reference for all Aural UI JavaScript methods. All methods are available through the global Aural object.

Toast

Aural.showToast(message, type, title, duration)

Display a toast notification with customizable message, type, title, and duration.

Parameters

Parameter Type Description
message string The main message to display in the toast
type (optional) 'success' | 'error' | 'warning' | 'info' The type of toast notification. Defaults to 'info'
title (optional) string Optional title to display above the message
duration (optional) number How long to show the toast in milliseconds. Defaults to 3000
Returns
void

Example

// Simple toast Aural.showToast('Operation successful'); // Success toast with title Aural.showToast( 'Your changes have been saved', 'success', 'Success' ); // Error toast with custom duration Aural.showToast( 'Failed to connect to server', 'error', 'Error', 5000 );

Chips (Tags Input)

Aural.initChips(containerId, options)

Initialize a chips (tags) input component. Returns an object with methods to interact with the chips.

Parameters

Parameter Type Description
containerId string The ID of the container element
options (optional) object Configuration options
options.maxTags (optional) number Maximum number of tags allowed
options.allowDuplicates (optional) boolean Whether to allow duplicate tags
options.onAdd (optional) function Callback when a tag is added
options.onRemove (optional) function Callback when a tag is removed
Returns
object - { getTags(), addTag(tag), clearTags() }

Example

// Initialize chips with options const chips = Aural.initChips('tags-input', { maxTags: 5, allowDuplicates: false, onAdd: (tag) => console.log('Added:', tag), onRemove: (tag) => console.log('Removed:', tag) }); // Get all tags const tags = chips.getTags(); // Add a tag programmatically chips.addTag('javascript'); // Clear all tags chips.clearTags();

Snackbar

Aural.showSnackbar(message, type, options)

Display a snackbar notification at the bottom of the screen with optional action button.

Parameters

Parameter Type Description
message string The message to display
type (optional) 'default' | 'success' | 'error' | 'warning' | 'info' The type of snackbar. Defaults to 'default'
options (optional) object Additional options
options.duration (optional) number How long to show the snackbar in milliseconds
options.action (optional) object Action button configuration { text, onClick }
Returns
void

Example

// Simple snackbar Aural.showSnackbar('Item deleted'); // Snackbar with action Aural.showSnackbar('Message sent', 'success', { duration: 5000, action: { text: 'Undo', onClick: () => console.log('Undo clicked') } });
