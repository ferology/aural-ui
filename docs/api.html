JavaScript API Reference

Complete reference for all Aural UI JavaScript methods. All methods are available through the global Aural object.

Toast Notifications

Aural.showToast(message, type, title, duration)

Display a toast notification with customizable message, type, title, and duration.

Parameters:
  • message (string) - The main message to display
  • type (string, optional) - 'success', 'error', 'warning', or 'info'. Defaults to 'info'
  • title (string, optional) - Optional title to display above the message
  • duration (number, optional) - How long to show the toast in milliseconds. Defaults to 3000
Example: 
// Simple toast
Aural.showToast('Operation successful');

// Success toast with title
Aural.showToast('Your changes have been saved', 'success', 'Success');

// Error toast with custom duration
Aural.showToast('Failed to connect to server', 'error', 'Error', 5000);

Modals

Aural.initModals()

Initialize all modals on the page. Call this once when the page loads.

document.addEventListener('DOMContentLoaded', () => {
  Aural.initModals();
});

Aural.openModal(modalId)

Programmatically open a specific modal by its ID.

Parameters:
  • modalId (string) - The ID of the modal element to open
Aural.openModal('my-modal');

Aural.closeModal(modalId)

Programmatically close a specific modal by its ID.

Parameters:
  • modalId (string) - The ID of the modal element to close
Aural.closeModal('my-modal');

Aural.toggleModal(modalId)

Toggle a modal's open/closed state.

Parameters:
  • modalId (string) - The ID of the modal element to toggle
Aural.toggleModal('my-modal');

Progress

Aural.setProgress(elementId, percentage)

Set the progress value of a progress bar.

Parameters:
  • elementId (string) - The ID of the progress element
  • percentage (number) - The progress percentage (0-100)
Aural.setProgress('upload-progress', 75);

Aural.setIndeterminate(elementId, indeterminate)

Set whether a progress bar should be in indeterminate state (animated loading state).

Parameters:
  • elementId (string) - The ID of the progress element
  • indeterminate (boolean) - Whether to show indeterminate animation

Snackbar

Aural.showSnackbar(message, type, options)

Display a snackbar notification at the bottom of the screen with optional action button.

Parameters:
  • message (string) - The message to display
  • type (string, optional) - 'default', 'success', 'error', 'warning', or 'info'
  • options (object, optional) - Additional options: { duration, action: { text, onClick } }
// Simple snackbar
Aural.showSnackbar('Item deleted');

// Snackbar with action
Aural.showSnackbar('Message sent', 'success', {
  duration: 5000,
  action: {
    text: 'Undo',
    onClick: () => console.log('Undo clicked')
  }
});
