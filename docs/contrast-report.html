Quick Fixes

1. Fix Dark Theme (dark.css)

/* Change muted text color */
--color-text-muted: #7a7a8a;  /* was #707080 */

/* For primary buttons, use dark text */
--color-button-primary-text: var(--color-bg-primary);  /* Use dark text instead of white */

2. Fix Light Theme (light.css)

/* Use darker green for light theme */
--primary-400: #228153;  /* was #5ebd8f - much better contrast on white */

/* OR keep the original green but override for light backgrounds */
--primary-light-accessible: #228153;

/* Fix muted text */
--color-text-muted: #6a717d;  /* was #9ca3af */

Alternative: Adaptive Primary Colors

Create theme-specific primary colors that maintain the green brand while ensuring accessibility:

/* dark.css - Can use lighter green */
--primary-400: #5ebd8f;  /* Original green - works great on dark */

/* light.css - Must use darker green */
--primary-400: #3d8a64;  /* Darker green - 4.5:1 on white backgrounds */

/* This way each theme is optimized for its context */
