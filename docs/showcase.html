Aural UI

A modern, accessible design system built for flexibility and customization

Aural UI is a comprehensive component library that combines beautiful design with practical functionality. Built on a foundation of design tokens and CSS custom properties, it offers unparalleled customization while maintaining consistency across your entire application.

Token-Driven

Customize every aspect of the design system with CSS custom properties. Change colors, spacing, typography, and more without touching component code.

Accessibility First

Built with WCAG AA compliance, keyboard navigation, proper ARIA attributes, and screen reader support out of the box.

Lightweight

Zero dependencies, vanilla JavaScript for interactive components, and optimized CSS for fast load times and smooth performance.

20+ Components

From basic buttons and inputs to advanced components like modals, dropdowns, and accordions - everything you need to build modern interfaces.

Dark Mode Ready

Switch between light and dark themes seamlessly. All components are designed to look great in both themes.

Developer Friendly

Simple, semantic HTML with intuitive class names. Copy-paste ready code snippets with clear documentation.

How to Use

Installation

Include the CSS and JavaScript files in your HTML. Choose a theme (dark or light) based on your preference. 

<!-- Include Aural UI CSS -->
<link rel="stylesheet" href="path/to/aural-ui.css">

<!-- Choose your theme (dark or light) -->
<link rel="stylesheet" href="path/to/dark.css">
<!-- OR -->
<link rel="stylesheet" href="path/to/light.css">

<!-- Include JavaScript for interactive components -->
<script src="path/to/aural-ui.js"></script>

Using Components

All components use simple, semantic HTML with intuitive class names. Here's an example button:

Responsive Grid System

Aural UI includes a powerful, mobile-first grid system for creating responsive layouts with ease. 

<!-- Container -->
<div class="container">
  <!-- Content with max-width and responsive padding -->
</div>

<!-- Basic Grid (1 column on mobile, 3 on desktop) -->
<div class="grid grid-cols-1 md:grid-cols-3 gap-6">
  <div>Column 1</div>
  <div>Column 2</div>
  <div>Column 3</div>
</div>

<!-- Auto-responsive grid (no media queries needed) -->
<div class="grid-cards">
  <div class="card">Card 1</div>
  <div class="card">Card 2</div>
  <div class="card">Card 3</div>
</div>

<!-- 12-column grid with custom spans -->
<div class="grid grid-cols-12 gap-4">
  <div class="col-span-12 md:col-span-8">Main content (8 cols on desktop)</div>
  <div class="col-span-12 md:col-span-4">Sidebar (4 cols on desktop)</div>
</div>

<!-- Common patterns -->
<div class="grid-2-col">...</div>  <!-- 2 columns on tablet+ -->
<div class="grid-3-col">...</div>  <!-- 3 columns on desktop -->
<div class="grid-4-col">...</div>  <!-- 2 on tablet, 4 on desktop -->

Breakpoints:

  • sm: 640px (mobile landscape)
  • md: 768px (tablet)
  • lg: 1024px (desktop)
  • xl: 1280px (large desktop)

Best UX Practices:

  • Use .container for content with max-width (better readability)
  • Use .grid-cards for auto-responsive card layouts (no breakpoints needed)
  • Stack content on mobile with grid-cols-1, then expand on larger screens
  • Use gap-4 (16px) for comfortable spacing, gap-6 (24px) for more breathing room
  • For complex layouts, use the 12-column grid with custom spans

Using Icons

Aural UI is designed to work seamlessly with Lucide Icons, a free and open-source icon library (MIT License). Simply include the Lucide script and use the data-lucide attribute. 

<!-- Include Lucide Icons -->
<script src="https://unpkg.com/lucide@latest"></script>

<!-- Initialize icons when DOM is loaded -->
<script>
  document.addEventListener('DOMContentLoaded', () => {
    lucide.createIcons();
  });
</script>

<!-- Use icons in your HTML -->
<i data-lucide="heart" style="width: 20px; height: 20px;"></i>
<i data-lucide="star" style="width: 24px; height: 24px; color: var(--color-warning);"></i>

<!-- Icons in buttons -->
<button class="btn btn-primary">
  <i data-lucide="download" style="width: 18px; height: 18px;"></i>
  Download
</button>

Customization with Tokens

All components are built with CSS custom properties (design tokens), making customization simple and powerful. Override tokens in your own CSS or use the live token customizer in the sidebar. 

:root {
  /* Override any token */
  --color-primary: #your-brand-color;
  --radius-md: 12px;
  --space-4: 20px;
  --font-sans: 'Your Font', sans-serif;
}

Accessibility

All components follow WCAG AA accessibility guidelines and include:

  • Keyboard navigation support (Tab, Arrow keys, Enter, Escape)
  • Visible focus indicators with proper contrast
  • Proper ARIA attributes and roles
  • Screen reader friendly labels and announcements
  • Minimum 44px touch targets for interactive elements
  • Respect for user's reduced motion preferences
