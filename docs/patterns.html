Empty State

Helpful placeholder when no data exists, with clear call-to-action.

No messages yet

You don't have any messages in your inbox. When you receive messages, they'll appear here.

Usage Tips:

  • Use a large, relevant icon to illustrate the empty state
  • Provide a clear, friendly message explaining why it's empty
  • Include a prominent call-to-action button
  • Keep the message positive and helpful, not discouraging

Form with Validation

Form showing success and error states with helpful validation messages.

Create Account

Username is available
Please enter a valid email address
Password must contain:
  • At least 8 characters
  • One uppercase letter
  • One number

Usage Tips:

  • Use colored borders to indicate validation states (green for success, red for error)
  • Show validation messages with icons for better recognition
  • For password fields, show requirements and check them off as user types
  • Validate on blur or submit, not on every keystroke
