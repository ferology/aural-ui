Flowing Wave Lines

A lightweight, animated flowing wave lines background built with canvas 2D. Smooth flowing curves with glowing particles!

  • Pure JavaScript with canvas 2D (no WebGL complexity)
  • Smooth flowing wave lines with multiple harmonics
  • Glowing particles following the wave paths
  • Fully customizable colors and wave patterns
  • Responsive and performant
  • Easy to integrate

<!-- Simple HTML Setup -->
<div id="wave-background" class="particle-background"
     data-particle-background>
</div>

<!-- Include CSS and JS -->
<link rel="stylesheet" href="particle-background.css">
<script src="particle-background.js"></script>

<!-- Or use JavaScript API -->
<script>
const flowingWaves = new ParticleBackground({
    containerId: 'wave-background',
    lineColor: 'rgba(147, 51, 234, 0.6)',
    particleColor: 'rgba(200, 150, 255, 0.8)',
    lineWidth: 2,
    lineCount: 15,
    waveAmplitude: 80,
    waveSpeed: 0.01,
    particleCount: 6
});
</script>
