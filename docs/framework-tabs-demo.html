Framework Tabs - Carbon Design System Style

Modern, professional tab design inspired by IBM's Carbon Design System, featuring smooth transitions, clean borders, and enhanced accessibility.

Design Features
  • Clean underline active indicator with smooth animation
  • Subtle hover states with background transitions
  • Carbon-inspired cubic-bezier timing functions (0.2, 0, 0.38, 0.9)
  • Framework-specific icons for visual identification
  • Responsive design with horizontal scrolling on mobile
  • Enhanced focus states for keyboard navigation
  • Smooth panel transitions with fade-in animations
Interactive Demo

Vanilla JavaScript - Pure HTML and JavaScript implementation

<!-- Simple, no build step required -->
<button class="btn btn-primary">Click me</button>
<button class="btn btn-secondary">Cancel</button>
<button class="btn btn-danger">Delete</button>

<script>
  // Add event listeners
  document.querySelector('.btn-primary').addEventListener('click', () => {
    console.log('Button clicked!');
  });
</script>

React - Component-based with hooks and props

import React from 'react';
import { Button } from '@aural-ui/react';

const MyComponent = () => {
  const handleClick = () => {
    console.log('Button clicked!');
  };

  return (
    <div>
      <Button variant="primary" onClick={handleClick}>
        Click me
      </Button>
      <Button variant="secondary">Cancel</Button>
      <Button variant="danger">Delete</Button>
    </div>
  );
};

export default MyComponent;

Vue - Template-driven with reactive data binding

<template>
  <div>
    <Button variant="primary" @click="handleClick">
      Click me
    </Button>
    <Button variant="secondary">Cancel</Button>
    <Button variant="danger">Delete</Button>
  </div>
</template>

<script>
import { Button } from '@aural-ui/vue';

export default {
  components: { Button },
  methods: {
    handleClick() {
      console.log('Button clicked!');
    }
  }
};
</script>

Svelte - Compiled, minimal runtime overhead

<script>
  import { Button } from '@aural-ui/svelte';

  function handleClick() {
    console.log('Button clicked!');
  }
</script>

<div>
  <Button variant="primary" on:click={handleClick}>
    Click me
  </Button>
  <Button variant="secondary">Cancel</Button>
  <Button variant="danger">Delete</Button>
</div>
CSS Implementation Details

The new tab styles are defined in page-common.css and include:

  • Smooth cubic-bezier transitions matching Carbon Design System
  • 3px underline for active state with slide-in animation
  • Framework-specific icon indicators (::before pseudo-elements)
  • Horizontal scroll on mobile with hidden scrollbars
  • Enhanced accessibility with proper ARIA attributes
  • Reduced motion support for accessibility
Keyboard Navigation

Try navigating with your keyboard: Use Tab to focus tabs, Enter or Space to activate, and Shift+Tab to go back. Notice the clear focus indicators!

Compatibility

The existing JavaScript in doc-utils.js remains fully compatible. No changes needed to the tab switching logic - all functionality is preserved while the visual design has been completely modernized.
