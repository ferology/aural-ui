Examples

Real-world patterns and templates you can use in your projects. Each example demonstrates best practices with Aural UI components.

Login Form

A modern, accessible login form with email and password inputs.

Sign In

Code

<form>
  <div class="form-group">
    <label for="email">Email address</label>
    <input type="email" id="email" class="input" placeholder="you@example.com">
  </div>

  <div class="form-group">
    <label for="password">Password</label>
    <input type="password" id="password" class="input" placeholder="••••••••">
  </div>

  <div class="form-group">
    <label class="checkbox">
      <input type="checkbox">
      <span>Remember me</span>
    </label>
  </div>

  <button type="submit" class="btn btn-primary" style="width: 100%;">
    Sign in
  </button>
</form>

Alert Messages

Different types of alert messages for various scenarios.

Success! Your changes have been saved.
Info: New features are available in the latest update.
Warning: Your session will expire in 5 minutes.
Error: Failed to process your request. Please try again.

Code

<div class="alert alert-success">
  <strong>Success!</strong> Your changes have been saved.
</div>

<div class="alert alert-info">
  <strong>Info:</strong> New features are available.
</div>

<div class="alert alert-warning">
  <strong>Warning:</strong> Your session will expire soon.
</div>

<div class="alert alert-error">
  <strong>Error:</strong> Failed to process request.
</div>

Settings Panel

A settings panel with toggle switches for user preferences.

Notification Settings

Email Notifications
Receive email updates about your account

Receive push notifications on your device
SMS Notifications
Receive text messages for important updates

Code

<div class="card">
  <h3>Notification Settings</h3>

  <div style="display: flex; justify-content: space-between; align-items: center;">
    <div>
      <div>Email Notifications</div>
      <div>Receive email updates about your account</div>
    </div>
    <label class="switch switch-only">
      <input type="checkbox" class="switch__input" role="switch" checked>
      <div class="switch__track">
        <div class="switch__thumb"></div>
      </div>
      <span class="switch__label"></span>
    </label>
  </div>

</div>

Data Table

A simple data table for displaying structured information.

Name Email Status Actions
John Doe john@example.com Active
Jane Smith jane@example.com Pending
Bob Johnson bob@example.com Inactive

Code

<table class="table">
  <thead>
    <tr>
      <th>Name</th>
      <th>Email</th>
      <th>Status</th>
      <th>Actions</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>John Doe</td>
      <td>john@example.com</td>
      <td><span class="badge badge-success">Active</span></td>
      <td><button class="btn btn-sm btn-secondary">Edit</button></td>
    </tr>
  </tbody>
</table>
