When to Use Each Font

System Sans (.font-sans)

  • Body text and paragraphs
  • UI labels and descriptions
  • Form inputs and buttons
  • Navigation menus
  • General interface text
<p class="font-sans">Body text</p>

Monospace (.font-mono)

  • Code snippets and blocks
  • API references
  • Terminal output
  • Tabular data and numbers
  • File paths and URLs
<code class="font-mono">const x = 5;</code>

Space Grotesk (Neon Theme)

  • Modern tech interfaces
  • Dashboard headings
  • Cyberpunk aesthetics
  • Gaming UI elements
  • Futuristic applications
<h1 style="font-family: 'Space Grotesk'">

Orbitron (.font-display)

  • Hero section titles
  • Brand logos (sparingly)
  • Special call-to-actions
  • Sci-fi themed headers
  • Attention-grabbing text
<h1 class="font-display">CYBER 2077</h1>

JetBrains Mono (Neon Code)

  • Code editors
  • Technical documentation
  • Developer tools
  • Terminal interfaces
  • JSON/XML display
<pre class="font-mono"><code>...</code></pre>

Helvetica Neue (Kinetic)

  • Brutalist editorial design
  • Bold, impactful headers
  • Minimalist interfaces
  • Swiss design aesthetic
  • Professional portfolios
<h1 style="font-family: 'Helvetica Neue'">
