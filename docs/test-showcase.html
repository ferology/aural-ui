What's New in This Release

Phase 1: Utilities

150+ new utility classes including backdrop filters, gradients, aspect ratios, dark mode, and RTL support.

Phase 2: Components

Enhanced buttons, inputs, modals, and tables with icon-only variants, character counters, and expandable rows.

Phase 3: Themes

High Contrast and Colorblind-Friendly themes for enhanced accessibility.

Phase 1: New Utilities

Over 150 utility classes have been added for rapid UI development. These utilities follow a Tailwind-like naming convention and are fully documented in the Utilities page.

Backdrop Filters

Perfect for glassmorphism effects on modals, cards, and overlays. Apply blur, brightness, contrast, and saturation to elements behind transparent surfaces.

Gradients

Create beautiful gradients with 8 directions (to-t, to-r, to-b, to-l, to-tr, to-br, to-bl, to-tl) and color stops (from-*, via-*, to-*) using all semantic colors.

Aspect Ratios

Maintain consistent aspect ratios for images, videos, and containers. Available ratios: aspect-square (1:1), aspect-video (16:9), aspect-wide (21:9), aspect-portrait (3:4).

Phase 2: Component Enhancements

Enhanced existing components with new variants and functionality. All examples shown here are now integrated into their respective component pages: Buttons, Inputs, Tables, and Modals.

Button Icon-Only

Square buttons optimized for icons. Available in 3 sizes: .btn-icon.btn-sm (36×36px), .btn-icon (44×44px), .btn-icon.btn-lg (52×52px). Perfect for toolbars and compact UIs. View full docs →

Input with Icons

Add prefix or suffix icons to inputs using .input-group wrapper. Supports all input sizes and both left/right icon positions. View full docs →

Character Counter

Live character counting with automatic warning (80%) and error (100%) states. Use .input-with-counter wrapper with .input-counter element. View full docs →

Characters: 0 / 100

Table Expandable Rows

Click rows to reveal additional details with smooth expand/collapse animation. Chevron icon rotates 90° when expanded. Use .expandable and .expanded-content classes. View full docs →

Phase 3: Theme System

Two new accessibility-focused themes have been added. Use the theme switcher buttons in the top-right to test each theme instantly.

High Contrast Theme

WCAG AAA compliant with 7:1+ contrast ratios. Pure black backgrounds, bright saturated colors, and enhanced 3px focus outlines. Perfect for users with low vision.

Colorblind-Friendly Theme

Optimized for deuteranopia, protanopia, and tritanopia. Uses blue for success (not green) and orange for errors (not red). Includes border indicators for additional clarity.
